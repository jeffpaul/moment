# Project Moment: Hosted Moment Concept

## Purpose

This is an optional future product path, not the required build strategy for the first prototype.

Moment should start as an open source WordPress plugin concept and private prototype. If the experience creates strong user, builder, or host demand, a host-supported Moment plan could become a compelling way to bring social-first creators into WordPress.

## Concept

A hosted Moment plan is not cheap WordPress hosting.

It is a consumer publishing product powered by WordPress.

The user starts by publishing, not configuring.

## Example User Journey

1. User chooses a handle or site address.
2. User picks a simple personal profile style.
3. User connects optional social destinations.
4. User skips or connects optional AI Assist.
5. User lands directly in Moment.
6. User publishes the first Moment.
7. The site is live.

The first post is the first site.

## Why This Could Matter to Hosts

Moment could give hosts a lower-friction consumer entry point:

- A simple $5/month-style personal publishing tier.
- A path for social-first creators who do not yet think of themselves as website owners.
- A habit-forming product based on frequent publishing rather than occasional site maintenance.
- A clean upgrade path into full WordPress when the user wants more control.
- Same content, same site, more capabilities unlocked over time.

## What It Should Not Be

The hosted Moment concept should not distort the open source product direction.

It should not require:

- Proprietary content storage.
- A single host.
- A single AI provider.
- A single social network.
- Lock-in to a hosted service.
- A custom post type solely for business packaging.

## Relationship to the Plugin

The `moment` plugin should remain useful on a normal self-hosted WordPress site.

A host-supported product could add:

- Seamless account creation.
- Site provisioning.
- Domain or subdomain setup.
- Authentication shortcuts.
- Built-in connector configuration.
- Media storage/scaling.
- PWA/home-screen onboarding.
- Upgrade prompts into full WordPress plans.

The plugin proves the experience.

A host can later reduce onboarding friction.

## Suggested Host Pitch

> Moment gives WordPress a simpler front door for people who publish from their phones. They can start by posting to their own site, then grow into the rest of WordPress when they are ready.

## Open Questions

- Should a hosted Moment plan use subdomains, connected domains, or both?
- Should social connectors be bundled, recommended, or user-selected?
- Should AI Assist be included, usage-limited, or entirely user-provided?
- What minimum theme/profile options are enough for launch?
- What upgrade moment best introduces the full WordPress experience without overwhelming the user?
